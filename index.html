How to Join Minorio

  1. Fork this repository: Minorio Members.
  2. Clone your forked repository to your local machine:
    3. git clone https://github.com/your-username/minorio-members.git
  3. Navigate to the members directory and create a new file named your-github-username.md.
  4. Open the file and fill in your details using this template:
    5. ---
name: Your Full Name
github: YourGitHubUsername
bio: Write a short bio about yourself (e.g., what you love or what you're learning!)
interests: [list, of, your, interests]
---
  5. Save the file and commit your changes:
    6. git add members/your-github-username.md
git commit -m "Add my profile to Minorio Members"
  6. Push your changes back to your forked repository:
    7. git push origin main
  7. Go to your forked repository on GitHub and create a pull request to the original repository.
