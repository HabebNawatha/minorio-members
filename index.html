Welcome to the Minorio Members Community

Register to join our growing community by following the steps below!

How to Join

  1. Fork this repository: Minorio Members.
  2. Clone the forked repository to your local machine.
  3. Navigate to the members directory and create a new file named your-github-username.md.
  4. Use the following template to fill out your information: 
    ---
name: Your Full Name
github: YourGitHubUsername
bio: A short bio about yourself
interests: [list, of, interests]
---
  5. Commit your changes and push them to your forked repository.
  6. Submit a pull request to the original repository.
